# Totem PostHog Demo

Projeto de exemplo utilizando React, Vite e TypeScript.

## Desenvolvimento

1. Instale as dependências
   ```
   pnpm install
   ```
   (ou use npm/yarn)

2. Inicie o servidor de desenvolvimento
   ```
   pnpm dev
   ```

3. Build de produção
   ```
   pnpm build
   ```

4. Pré-visualização
   ```
   pnpm preview
   ```

As variáveis de ambiente estão em `.env.example`.
