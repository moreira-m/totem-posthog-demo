# Totem PostHog Demo

Projeto de exemplo utilizando React, Vite e TypeScript.

## Desenvolvimento

1. Copie o arquivo de variáveis de ambiente
   ```
   cp .env.example .env
   ```

2. Instale as dependências
   ```
   pnpm install
   ```
   (ou use npm/yarn)

3. Inicie o servidor de desenvolvimento
   ```
   pnpm dev
   ```

4. Build de produção
   ```
   pnpm build
   ```

5. Pré-visualização
   ```
   pnpm preview
   ```
